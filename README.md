# rustv1

A small collection of Rust learning examples and simple Cargo projects.

**Prerequisites**
- Install Rust via `rustup` (https://rustup.rs).
- Use a terminal that supports PowerShell on Windows.

**Quick Start**
- Run a single `.rs` example:
  - Compile: `rustc 0-hello.rs`
  - Run: `.\0-hello.exe`
- Use Cargo for the `project1` app:
  - Build & run: `cd project1; cargo run`
  - Build only: `cd project1; cargo build`
